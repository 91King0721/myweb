# 整学期空闲教室查询系统

一个查看全校教室空闲情况的交互式课表查询工具，支持收藏、折叠、自定义时间段筛选，主要内容由CC编写。

> **当前版本:** v0.1 — [在线体验](https://91king0721.github.io/myweb/)

---

## 版本历史

### v0.1（当前版本）
新增三大功能，全面提升查询效率：

**✨ 收藏功能**
- 点击教室旁的星标即可收藏，数据保存在浏览器本地
- 控制栏新增"收藏"切换按钮，一键筛选仅看已收藏教室
- 已收藏教室行高亮显示，易于识别

**📅 折叠式日程**
- 一周五天改为手风琴式折叠，默认展开当天
- 点击任意天标题展开/收起，每次仅展开一天，界面更清爽
- 进入页面自动定位到当前周次和当天

**⏱ 时间段筛选**
- 预设三个时段：上午(1-4节)、下午(5-8节)、晚上(9-12节)
- 新增**自定义时段**：自由选择起始和结束节次（如 1-9、5-12、3-6）
- 表格仅显示该时段列，空闲计数实时更新

> 完整变更清单见下方 [v0.0 → v0.1 变更说明](#v00--v01-变更说明)

## 功能

- **欢迎页**：一级菜单入口，提供"查询空闲教室"入口
- **教室查询**：按周次和教学楼筛选，支持教室名称搜索
- **时间段筛选**：预设上午/下午/晚上 + 自定义任意节次范围
- **教室收藏**：星标收藏常用教室，一键过滤查看
- **折叠日程**：手风琴式展开收起，默认展开当天
- **空闲统计**：每节课占用/空闲情况一目了然
- **响应式设计**：完美适配桌面端、平板和手机

## 本地使用

直接在浏览器打开 `index.html` 即可，无需任何构建工具或依赖。

## 部署

部署到 GitHub Pages：

```bash
git push origin main --tags
```

然后在 GitHub 仓库的 Settings > Pages 中选择 `main` 分支，根目录 `/` 即可。

### 详细功能列表

**收藏系统**
- `localStorage` 存储，key 格式 `"教学楼|教室名"` 确保跨楼同名教室可区分
- 星标 SVG 点击切换，事件委托处理（兼容 SVG 子元素点击）
- 收藏过滤开关，激活时仅显示已收藏教室

**折叠式日程**
- 单例展开模式（同时仅展开一天）
- 点击折叠图标或整行均可切换
- 页面加载自动展开当天

**时间段筛选**
- 三个预设时段组：1-4节（08:00-12:00）、5-8节（14:00-18:00）、9-12节（19:00-22:30）
- 自定义时段：选择起始和结束节次，支持任意范围（自动处理起止颠倒）
- 过滤器流水线：文本搜索 → 收藏过滤 → 时段过滤

**时间段修正**
- 将原独立45分钟小节时间改为双节次配对时间
- 每对节次显示起止时间（如 1节=08:00、2节=09:50 对应 1-2节 08:00-09:50）

### 技术细节
- `getCurrentWeek()` 从 2026年3月9日（第1周周一）推算当前周次，自动适配学期范围 1-18周
- `getCurrentDayName()` 自动判断当天星期，周末默认展开周一
- 选中收藏行通过 `tr.fav-row` 高亮（#fffbeb / 深色模式 #2a2510）
