# 整学期教室课表

一个查看全校教室空闲情况的交互式课表查询工具。

## 功能

- 按星期查看所有教室的课程安排
- 根据教室名称搜索过滤
- 直观展示每节课的空闲/占用状态
- 显示每天的教室空闲数量统计

## 部署

部署到 GitHub Pages：

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/<你的用户名>/<仓库名>.git
git push -u origin main
```

然后在 GitHub 仓库的 Settings > Pages 中选择 `main` 分支，选择根目录 `/` 即可。
